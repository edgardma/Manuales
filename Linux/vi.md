## Comandos para VI:

Los principales comandos de VI:

| Comando   | Descripción                                                      |
| --------- | ---------------------------------------------------------------- |
| *:q*      | Salir del editor.                                                |
| *:q!*     | Salir sin guardar los cambios.                                   |
| *:wq*     | Salir y guardar los cambios.                                     |
| *:wq!*    | Salir y guardar los cambios, pero no verifica si se guardó o no. |
| *:set nu* | Mostrar el número de las líneas.                                 |
